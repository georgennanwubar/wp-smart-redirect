# 📦 Smart Redirect For Returning Users – Version 2.0

Smart redirection plugin for WordPress that detects returning users by cookie and routes them with precision. Now with full Google Analytics (gtag & GA4) support, redirection tracking, and admin controls.

## ✨ Features
- Detect returning users using cookies
- Redirect to any URL (e.g., dashboard, app, etc.)
- Full Google Analytics event tracking (gtag.js)
- GA4 server-side fallback with Measurement Protocol
- WP Admin settings (no coding required)
- IP + timestamp logs with CSV export
- Role-based redirection exclusion
- Admin email alerts after X redirects

## 🛠 Installation
Upload the ZIP to `/wp-content/plugins/` and activate it.
Then go to **Settings → Smart Redirect** to configure.
