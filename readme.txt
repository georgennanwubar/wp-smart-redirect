Redirect returning users based on cookie detection, with full Google Analytics + GA4 support, redirection tracking, logs and more.

== Description ==

This plugin redirects returning users based on the presence of a specific cookie. Perfect for SaaS platforms, eLearning sites, gated portals or membership dashboards.

**Core Features:**
* Cookie-based redirection logic
* Role-based user exclusion
* Google Analytics (gtag) + GA4 Measurement Protocol fallback
* Redirection log + export (timestamp & IP)
* Admin email notifications on milestones
* Full WP admin UI (no code required)

== Screenshots ==
1. Settings Panel: Configure cookie, URL, GA & exclusions
2. Redirection Logs: Track timestamp and IP
3. Export Logs & Stats Overview

== Installation ==

1. Upload to `/wp-content/plugins/`
2. Activate via WordPress admin
3. Configure via **Settings → Smart Redirect**
