Smart Redirect For Returning Users (v1.0)

What It Does

This plugin checks for a specific cookie on your site visitors’ browsers. If the cookie exists and matches your defined value, it redirects the user to a custom URL (like an app dashboard or member area).

Getting Started

Toggle the 'Invert Logic' option if you want the redirect to run on all pages except those listed.

Only apply this plugin on specific pages by entering one or more full URLs in the provided field. For example, enter your homepage URL to restrict it to the homepage only.

  1. Install and activate the plugin from your WordPress dashboard.
  2. Navigate to Settings → Smart Redirect.
  3. Enter the cookie name (e.g., smart_user_cookie) and expected value (e.g., true).
  4. Enter the destination URL (e.g., https://yourapp.com/dashboard).
  5. Select any roles to exclude (e.g., Admin, Editor).
  6. Save your settings. The plugin works immediately.

Tracking Options

Optionally, you can enable Google Analytics tracking. You may enter your GA4 Measurement ID and API Secret to track events both client-side and server-side.

Log Exporting

Every redirect is logged with a timestamp and IP address. Logs are viewable in the plugin settings and can be exported as CSV.

Advanced Settings

To automatically remove all plugin settings on uninstall, enable the 'Delete Data On Uninstall' setting in the admin panel.
